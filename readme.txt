Adding Share Buttons as Overlay on YouTube Videos

== Description ==

Adding Share Buttons as Overlay on YouTube Videos
Embed youtube with shortcode
Show share facebook, tweet when hover to YouTube video

== Installation ==

1. Upload `at-youtube-share-overlay` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Use shortcode [wpp-youtube id="dvRlxx2VSng"] for post, page, widget text, etc
